# Githubでの調べ方
## 開発言語指定
**[ワード] language:[言語]**  
e.g.) `async language:javascript`

## 拡張子指定
**[ワード] extension:[拡張子]**  
e.g.) `props extension:jsx`

## パス指定
**[ワード] path:[パス]**  
e.g.) `console path:app/public`

## ファイル指定
**[ワード] in:[ファイルパス]**  
e.g.) `RUN in:Dockerfile`

## ファイル名検索
**filename:[ファイル名]**  
e.g.) `filename:.vimrc`
